During my 5th college semester i was assigned to create my own encryption algorithm, at first, i was planning to just do just the bare minimum to get a passing grade, but as i was investigating more about modern cryptography, i got really into it, which made me really proud of myself. The final algorithm contains a fair share of techniques that are actually used, like scrambling the data, salting, bit manipulation, and a few more. I named it Antumbra because it sounds cool and i think data as being eclipsed.

I made a simple UI in Python to be able to test the algorithm, which was pretty easy to do. To summarize the way it works, you input the text you want to encrypt, then you input whatever key you want, and the text would be encrypted, to decrypt it, get the text and the same key. This data goes through like 7 steps of manipulation to encrypt it, you can learn more by checking the repository.
